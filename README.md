# HyponymyExtraction
HyponymyExtraction Demo based on pattern match and posfilter method，基于模式匹配与词性过滤的上下位语义抽取与展示
# 项目介绍
本项目主要是从构造显式上下位关系模式，并使用词性过滤的方式对上下位事件进行表示，从而绘制出上下位关系对
# 本项目的两个组成部分
１）7千多篇文学作品语料库的构建
２）基于文学作品语料库的上下位关系抽取及展示
